Percentage Calculation Formulas

Basic Percentage

Result = (Percentage / 100) √ó Number

Example: 25% of 80 = (25/100) √ó 80 = 20

Percentage of Total

Percentage = (Part / Total) √ó 100

Example: 20 is what % of 80? = (20/80) √ó 100 = 25%

Percentage Change

% Change = ((New - Old) / Old) √ó 100

Example: From 50 to 75 = ((75-50)/50) √ó 100 = 50%
