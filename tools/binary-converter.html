Number System Guide

  • Binary (Base 2): Uses digits 0 and 1. Common in computer science.
  • Decimal (Base 10): Standard number system using digits 0-9.
  • Hexadecimal (Base 16): Uses digits 0-9 and letters A-F.
  • Octal (Base 8): Uses digits 0-7. Less common but still used in computing.
